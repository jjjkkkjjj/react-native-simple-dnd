# Example

The simple example app using expo.

## Setup

Insall Expo.

```bash
npm install -g yalc
yalc publish react-native-simple-dnd
cd example
yalc add 
npm install -S
npx expo start
```

## Development

- When the [src codes](../src/) are updated,

```bash
npm run build
yalc push
```
